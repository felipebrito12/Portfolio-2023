Quem sou eu?

Meu nome é Felipe Brito Sousa e sempre tive interesse nas áreas de tecnologia. Ainda na adolescência tive meu primeiro contato com programação atráves do GameMaker Studio. Uma engine direcionada para criação de jogos 2D com linguagem própria. Apesar disso, meu caminho dentro da programação só começou realmente em julho de 2022, através do desenvolvimento web.


Atualmente mantenho meus estudos no Front-end e estou buscando minha primeira experiência profissional na área, mas com planos de futuramente adentrar no Back-end.

TECNOLOGIAS

HTML5, sigla para Hypertext Markup Language, é uma linguagem de marcação de hipertexto para apresentar e estruturar o conteúdo na web. Sendo uma das bases do desenvolvimento web.

Cascading Style Sheets, ou CSS, é um mecanismo para adicionar estilos a uma página web, aplicando design personalizados nas tags HTML.

SASS é um pré-processador CSS que adiciona novos recursos ao CSS básico, deixando o processo de estilização da página mais intuitiva e organizada. Após processado o SASS gera CSS comum.

JavaScript é uma linguagem de programação interpretada estruturada, sendo comumente utilizada para criar as interações entre o usuário e a página web


Juntamente com HTML e CSS, o JavaScript é uma das três principais tecnologias da World Wide Web.

GIT é um Sistema de Controle de Versões Distribuído. Estes sistemas de controle permitem registrar quaisquer alterações feitas em um código, armazenando essas informações e permitindo que o desenvolvedor possa regredir a versões anteriores de uma aplicação

O Bootstrap é uma estrutura de desenvolvimento web de código aberto e gratuita. Foi projetada para facilitar o processo de criação de sites responsivos para dispositivos móveis, fornecendo uma coleção de sintaxe para designs de modelo.
